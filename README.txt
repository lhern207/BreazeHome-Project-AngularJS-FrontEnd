This directory includes my contributions to the BreazeHome4.0 Project.

BreazeHome4.0 is a real estate website found at:
https://breazehome.com/

My contribution can be found on the results page:
https://breazehome.com/#/results

I implemented the filters on the results page, including the dynamically
updated density charts. 

The project's front end was developed using AngularJS.
Banckend was Django Rest API.
Database was PostgreSQL.
My contribution consisted mostly of front end changes in the files provided.
